🔬 Зертханалық жұмыс

📌 Тапсырма:

Берілген кодта қателер бар. Бағдарламаны толық жұмыс істейтін деңгейге жеткізіңіз.

  • Канкулятор жасау керек main.py қосу арқылы
  • Қателерді табу
  • Кодты рефакторинг жасау

📄 main.py
Главный py файл осыны қосу арқылы Канкулятор жұмыс жасау керек

import datetime import random a = input(a: ) b = input(b: ) print("1 - қосу") print("2 - азайту") print("3 - көбейту") print("4 - бөлу") choice = input("Таңдау: ") if choice == "1": print(add(a, b)) elif choice == "2": print(sub(a, b)) elif choice == "3": print(multiply(a, b)) elif choice == "4": print(divide(a, b))

📄 add_sub.py

from add_sub import add, sub def add(a, b): return a + b def sub(a, b): return a - b

📄 mul_div.py

from mul_div import multiply, divide def multiply(a, b): return a * b def divide(a, b): return a / b
