🐍 Python pdb — Толық түсіндірме

Debugger және қателерді табу құралы

📘 pdb деген не?

pdb — Python Debugger (түзеткіш). Ол программадағы қателерді табуға көмектесетін Python-ның ішкі debugger құралы.

Ол кодты қадам-қадаммен тоқтатып, айнымалылардың мәнін көруге мүмкіндік береді.

❓ Неге керек?

Мысалы программа дұрыс жұмыс істемей тұр:

a = 10 b = 0 result = a / b print(result)

Мұнда қате шығады:

ZeroDivisionError

Осындай кезде pdb арқылы қай жерде қате екенін тексереміз.

⚙️ pdb қолдану

Ең оңай тәсіл:

import pdb pdb.set_trace()

Кодтың ішіне осыны жазамыз.

🧠 Мысал

import pdb a = 10 b = 2 pdb.set_trace() result = a / b print(result)

🔎 Айнымалыны көру

p a

Нәтиже:

10

➡️ Келесі жолға өту

n

(next) → келесі код орындалады

📌 pdb.set_trace() қалай жұмыс істейді?

Python дәл сол жерге келгенде программа тоқтайды. Сосын сен тексересің.

1-жол ↓ n 2-жол ↓ n 3-жол

🧪 Толық көрініс

(Pdb) n -> c = a + b (Pdb) n -> print(c) (Pdb) n 7

⚙️ Debugger қалай жұмыс істейді?

Код жүріп келеді ↓ pdb.set_trace() ↓ Программа тоқтайды ↓ Сен тексересің ↓ n / c / p қолданасың

🔥 Ең көп қолданылатын командалар

p a → a мәнін көру n → келесі жол c → жалғастыру q → шығу l → кодты көру

💥 Қандай қателерді табады?

  • ZeroDivisionError
  • IndexError
  • TypeError
  • NameError
  • Логикалық қателер

📋 Мысал (locals)

import pdb a = 10 b = 20 name = "Ali" pdb.set_trace()
p locals()

Нәтиже:

{'a': 10, 'b': 20, 'name': 'Ali'}

📖 l — кодты көру

l

Мысал:

7 -> c = a + b 8 print(c)

🚀 c — жалғастыру

c

👉 Программаны тоқтаусыз соңына дейін жүргізеді

🚪 q — шығу

q
